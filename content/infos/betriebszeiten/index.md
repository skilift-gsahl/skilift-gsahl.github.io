|                               |                   |
| ----------------------------- | ----------------: |
| Mittwoch                      | 13.00 – 16.30 Uhr |

| Sonntag                       | 11.00 – 16.30 Uhr |
| Täglich während Schulferien   | 13.00 – 16.30 Uhr |

Die obere Liftanlage wird nur bei genügender Frequenz in Betrieb gesetzt.
